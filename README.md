# Sistema de Controle de Entregas

## Funcionalidades
- Login com senha (12345678)
- Cadastro de mercadoria
- Cadastro de endereço
- Vinculação 1 para 1 entre produto e endereço

## Fluxo Git utilizado
- Desenvolvimento feito na branch `release-dev`
- Nenhum commit direto na `main`
- Após finalização, foi feito Pull Request
- Merge realizado para a branch main

- HTML
- CSS
- JavaScript
